# Responsive vs Adaptive

- Simplicity is crucial
- Visual is important 
- Goals are important
- Responsive 
  - content reacting to size of user's device
- Adaptive
  - copy text
  - HTML
  - audio-visual
  - interactive
  - can adopt to context of use
- Hybridized
  - website looking like an app
- Geofencing / Eye beacon
  - giving content based on context
